기법 설명 및 유의사항

 

1.FCFS

(1)비선점

(2)프로세스는 준비 큐에 도착한 순서대로 CPU를 할당받음

유의 사항: 도착 시간이 같은 프로세스 경우 테이블에 입력한 위에 값부터 실행

2.SJF

(1)비선점

(2)실행시간이 가장 짧은 작업을 우선적으로 스케줄링

유의 사항: 서비스 시간이 같은 프로세스 일 경우 테이블에 입력한 위에 값부터 실행

3.HRN

(1)비선점

(2)응답률이 가장 높은 프로세스부터 할당함

유의 사항: 도착시간이 같은 프로세스 일 경우 테이블에 입력한 위에 값부터 실행

4.Round Robin

(1)선점

(2)FCFS 기법으로 프로세스를 처리하되, 프로세스에게 동일한 CPU 시간을 할당

유의 사항: 도착시간이 같은 프로세스 일 경우 테이블에 입력한 위에 값부터 실행

5.우선순위

(1)선점

(2)우선순위가 높은 프로세스를 우선적으로처리

유의 사항: 가)우선순위가 같은 프로세스 일 경우 테이블에 입력한 위에 값부터 실행

\t나)숫자가 낮을수록 우선순위는 높다

2.SJF

(1)선점

(2)잔여 실행시간이 가장 짧은 프로세스를 우선적으로 스케줄링

유의 사항: 서비스 시간이 같은 프로세스 일 경우 테이블에 입력한 위에 값부터 실행
